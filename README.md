# Interaction with DEP — Telegram-бот

Бот для заведения продуктов на проработку с загрузкой файлов в Google Drive.

## Возможности

- **Заведение продукта на проработку** — пошаговый flow:
  - Поставщик (выбор из списка или ввод нового)
  - Номенклатура поставщика (прайсовое название)
  - Ед. изм (шт, кг, упак)
  - Цена без НДС, ставка НДС (10%, 22%)
  - Загрузка сертификатов и деклараций
  - Фото продукта
  - Фото этикетки

- **Процесс проработки** — заглушка (будущая реализация)

## Запуск

```bash
# Установка зависимостей
pip install -r requirements.txt

# Переменные окружения
cp .env.example .env
# Заполните BOT_TOKEN (получить у @BotFather)

# Демо-поставщики (опционально)
PYTHONPATH=. python scripts/seed_suppliers.py

# Запуск бота
PYTHONPATH=. python run.py
```

## Структура проекта

```
bot/
  config.py          # Конфигурация
  main.py            # Точка входа
  handlers/          # Обработчики команд и диалогов
  keyboards/         # Клавиатуры
  models/            # SQLAlchemy-модели
  services/          # БД, Google Drive
```

См. [docs/google-drive-setup.md](docs/google-drive-setup.md). Без настройки Drive бот работает, но файлы не загружаются.
