# Инструкция: Первый запуск проекта

## Шаг 1. Установка зависимостей

Откройте терминал в корне проекта и выполните:

```bash
npm install
```

Убедитесь, что все пакеты установились без ошибок.

## Шаг 2. Проверка окружения

Настройте переменные окружения. Пример структуры `.env`:

```
API_KEY=your_api_key
DATABASE_URL=postgresql://localhost:5432/mydb
```

![Схема подключения к базе](https://picsum.photos/600/300?random=1)

*Рис. 1 — Схема подключения компонентов*

## Шаг 3. Запуск приложения

Выполните команду запуска:

```bash
npm run dev
```

Сервер должен быть доступен по адресу `http://localhost:3000`.
